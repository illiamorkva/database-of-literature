What is temporary list?

  • This page shows the temporary list of documents you can create during a Scopus session.
  • You can work with this list the same way you work with any search results list - output the list, track citations, refine the list, and so on.

To save documents to a saved list:

You must be logged in to save documents from your list or to work with your account Saved lists.
If you have previously saved lists for your account, you can access them after logging in from the main Lists menu or via My Scopus and clicking Saved lists.
